# jeevasethu

A new Flutter project.

🩸 JeevaSethu – Blood Donor Finder App

JeevaSethu is a mobile application designed to connect blood donors with people in need during medical emergencies. The app allows donors to register their details and enables users to search for available donors based on blood group.

Built using Flutter for cross-platform mobile development and Firebase Authentication for secure user login and registration.

🚀 Features

🩸 Search blood donors by blood group

👤 User registration and login system

🔐 Secure authentication using Firebase

📱 Clean and responsive mobile UI

⚡ Fast and reliable donor search

🧑‍🤝‍🧑 Easy donor registration

⚙️ Installation
1. Clone the repository

git clone https://github.com/yourusername/jeevasethu.git

cd jeevasethu

2. Install dependencies

flutter pub get

3. Setup Firebase

Create a project in Firebase Console

Add your Flutter app to the project

Download the Firebase configuration files

For Android:

Place google-services.json inside:

android/app/

For iOS:

Add GoogleService-Info.plist inside:

ios/Runner/

4. Enable Firebase Authentication

Go to Firebase Console → Authentication → Sign-in Method

Enable Email/Password Authentication

5. Run the application

flutter run

💡 How It Works

Users sign up or log in using Firebase Authentication

Donors register their blood group and contact information

Patients search for donors based on blood type

The application displays available donors instantly
